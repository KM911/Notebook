>[!tip] list open files
>displays information about open files on a Linux system

| flag | usage |
| ---- | ---- |
| `-i` | network interface |
| `-p` | check open file by pid |
| `-u` | by user |
>[!example]- check 3000 port usage 
>```bash
>lsof -i :3000 
>```
