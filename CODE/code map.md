>[!faq]- 第一门编程语言学什么 ? 
> 只学xx就足够了
> 假如某一门编程语言在所有领域都是无敌的存在,根本就不会有如此多的编程语言存在了.它们适合的人,适合的项目,适合的应用场景并不一样.只有在一个特定场景下,才有合适的解. 
> 你应该仔细区分,学习其中优秀的部分.  
> 所以你应该广泛涉猎.
