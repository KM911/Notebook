
>[!faq] 软件开发时
>我们希望可以知道用户是否已经安装了某些依赖的软件,如果检测呢? 

## 利用 version 

>[!note] `gcc --version`
>大部分的命令行程序,基本都有 `--version` 命令 , 我们通过读取该命令的退出码, 就可以知道是否安装有对应软件了.


>[!example]  
>```go
> if util.ExecuteSilent("clang++ --version") == 0 {
>     CC = "clang++ "
>     return
> }
>```

[[python环境检测]] : 利用[[CODE/PYTHON/python]]的 `try` 捕获缺少 lib的错误, 引导用户安装.
